Context & budget
How many recent messages your main AI sees while writing. 0 = off (it sees the whole chat, the normal way). With a number set, older history is trimmed and the memory sheet stands in for it — cheaper and more focused. Trimming only kicks in once the first memory sheet exists.
The newest N messages are never mined for facts — you might still swipe or edit them. The Memory Agent only sees them as clearly-labeled tentative context for planning. 4 is the default.
Every N settled messages are distilled into ONE "Story so far" sentence. Smaller = a more detailed story spine but more background LLM calls. 10 is the default.
The Memory Agent staples on a few EXTRA memories connected to the scene's facts. Instead of always the top-ranked ones, it wanders the memory graph at random — restaurant → Luigi's → first date — surfacing a different connected chain each time. This sets how many to collect. 0 turns it off.
On by default. Facts tagged with a who-knows list are hidden unless someone in the current scene (the main pair or a PRESENT-listed NPC) is on it, so secrets stay secret. Name matching is alias-aware — "Trish" and "Trish Mitchells" count as the same character. Turn OFF to let every character see every fact (secrets will leak) — useful if facts seem to go missing in a shared database.
Memory Agent — Extra Instructions
Appended to the background Memory Agent's task each run — e.g. genre-specific things to track. The built-in rulebook itself is fixed (cache-stable) and is not edited here.
Catch-up import (existing chats)
Got a long chat from before you turned this on? This reads back through the unprocessed backlog in chunks — one AI call per chunk of messages, resumable if you cancel — and rebuilds the memory sheet at the end. Don't chat while it runs.
Bigger chunks = fewer, cheaper calls but coarser notes. 8 is a good default.
Memory Agent prompt
The FULL system prompt the background extraction agent runs on — it covers only the fact-extraction and NEED-selection pass; the beat and sheet-head passes use fixed built-in prompts. Leave blank or Reset to use the built-in default. This is separate from the Memory tab's "extra instructions", which are still appended on top of whatever prompt is used here.
Reflect Agent prompt
The FULL system prompt for the periodic Reflect maintenance pass. Leave blank or Reset to use the built-in default.
Database Profiles
Save a copy of your whole memory, or load an old one back. Handy to share facts between characters or undo a mess. E.g. save a snapshot before a big risky scene so you can roll back.
